View Encapsulation

membatasi scope CSS sebuah component, agar style component tidak mempengaruhi component lain

3 tipe View Encapsulation:
  • 1. Emulated (sangat ter-isolasi, default angular)
  • 2. None (CSS bersifat global & tdk ada isolasi)
  • 3. ShadowDom (membaca global css namun css tetap terisolasi)
