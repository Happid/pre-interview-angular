12. Data Binding

menghubungkan data (component) dengan tampilan (template HTML)

  • 1. One-Way Binding
  • A. Interpolation {{" {{ }} "}} 
            
          // ts
          title = 'Halo Angular';

          // html
          {{"{{title}}"}}
  • B. Property Binding [property] 
            
          // ts
          isDisabled = true;

          // html
          <{{"button"}} {{"[disabled]=\"isDisabled\""}}>
            Submit
          <{{"/button"}}>
  • C. Event Binding (event) 
            
          // ts
          {{"onClick() { "}}
          {{" console.log('Button diklik'); "}}
          {{"}"}}

          // html
          <{{"button"}} {{"(click)=\"handleClick()\""}}>
            Click Me
          <{{"/button"}}>
